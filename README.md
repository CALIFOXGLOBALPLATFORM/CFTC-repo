# CFTC-repo
Build a production-ready full-stack web application called Commodities Futures Trading Commission.

Tech stack:

Frontend: React + Vite + TypeScript + Tailwind CSS

Backend: Node.js + Express

Database: PostgreSQL with Prisma ORM

Authentication: JWT + bcrypt

State management: React Query


Features:

Responsive landing page

User registration and login

Password reset

User dashboard

Investment plans

Deposit request page

Withdrawal request page

Transaction history

Notifications

Profile & settings

Referral dashboard

Admin dashboard

User management

Deposit approval

Withdrawal approval

Investment plan management

Reports and audit logs


Database:
Users, Admins, InvestmentPlans, Investments, Deposits, Withdrawals, Transactions, Notifications, Referrals, AuditLogs.

Design:
Modern dark navy blue, black, and gold trading theme with mobile, tablet, and desktop responsiveness.

Create a complete folder structure, REST API, Prisma schema, database migrations, protected routes, reusable React components, and production-ready configuration.
